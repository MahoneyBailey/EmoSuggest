# EmoSuggest: AI That Reads Your Face & Mind While You Type

**Revolutionary on-device iOS app** powered by **Apple Intelligence**, ARKit TrueDepth camera, and self-training Core ML models.

EmoSuggest analyzes micro-expressions, facial movements, typing rhythm, thought pauses, and context — delivering **hyper-personalized live word/phrase suggestions** that perfectly match your emotional state and intent.

## ✨ Features
- Real-time micro-emotion detection via TrueDepth
- Self-training models that adapt to your unique patterns
- Live predictive suggestions in any text field
- 100% on-device privacy (no data leaves your iPhone)
- Seamless integration with Apple Intelligence

## 🚀 Tech Stack
- SwiftUI + Foundation Models
- ARKit (TrueDepth blend shapes)
- Core ML (on-device self-training)
- App Intents

**iOS 18+ • TrueDepth device required**

## 📥 Quick Start
1. Clone the repo
2. Open in Xcode
3. Run on device & grant camera permissions
